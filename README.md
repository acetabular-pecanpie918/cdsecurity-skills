## 🔐 Security review focus

This project is meant for smart contract security work, so the checks should focus on:
- who can call each function
- how funds move through the contract
- whether external calls are safe
- whether state updates happen in the right order
- whether arithmetic and token logic behave as expected
- whether admin powers are too broad

Use the skills as part of a wider review process. Read the code yourself and test the findings before you make changes.
